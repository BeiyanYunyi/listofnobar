含义
----

listofnobar，可以是list of の bar（之吧清单）也可以是list of no bar（无吧清单）

一为本吧，二意流浪。

事记
----

2020年4月1日，百度贴吧“真夏夜之银梦吧”禁止发言。成员大部转移至“假冬日之金醒吧”。

2020年4月5日，“假冬日之金醒吧”在吧主上任前夕遭爆破。成员四散中。本repo由“夏日相扑大会”吧建立。
